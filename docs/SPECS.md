# Especificación del Producto

## Problema

El cerebro humano está diseñado para tener ideas, no para almacenar y reconstruir constantemente el contexto de trabajo. Sin embargo, el mundo actual exige hacer ambas cosas simultáneamente, generando:

- Sobrecarga cognitiva: Memoria saturada con información contextual
- Context switching constante: Pérdida de 23 minutos promedio por interrupción
- Fatiga mental: Decisión fatigue y agotamiento prematuro
- Errores recurrentes: Olvido de detalles importantes y duplicación de trabajo
- Sensación de estar siempre atrasado: Ansiedad por no recordar todo

## Solución

Sistema de gestión de conocimiento personal que actúa como "segundo cerebro", capturando y organizando automáticamente el contexto de trabajo para liberar la mente para tareas creativas y estratégicas.

## Usuario Objetivo

### Primario
- Knowledge workers: Desarrolladores, diseñadores, product managers
- Creadores de contenido: Escritores, investigadores, periodistas
- Estudiantes y académicos: Universitarios, investigadores, docentes

### Características del usuario
- Manejan múltiples proyectos simultáneos
- Necesitan recordar contexto complejo
- Valoran la productividad y el aprendizaje continuo
- Frustrados con herramientas actuales (demasiado rígidas o complejas)

## Feature Selection MVP

### Must Have (M)
1. Input de texto directo con guardado automático
2. Transcripción de audio a texto
3. Organización por proyectos/espacios
4. Búsqueda full-text básica
5. Vista de notas recientes
6. Export a Markdown

### Should Have (S)
7. Tags manuales para categorización
8. Vista de conexiones entre notas (grafo simple)
9. Búsqueda por fecha/rango
10. Templates de notas
11. Modo offline con sync

### Could Have (C)
12. Búsqueda semántica inteligente
13. Extracción automática de entidades
14. Resúmenes automáticos de notas largas
15. Vista timeline cronológica
16. Integraciones con Slack/Discord
17. Browser extension para captura rápida

### Won't Have (W) - Para v2
18. Colaboración en tiempo real
19. Versionado de notas
20. API pública
21. Mobile app nativa
22. Espacios de equipo/empresa
23. Analytics avanzados

## Propuesta de Valor

Para knowledge workers que necesitan gestionar múltiples contextos de trabajo, nuestro producto es un sistema de captura y organización de conocimiento que, a diferencia de Notion o Obsidian, prioriza la velocidad de captura y la simplicidad de uso sobre la complejidad de features.





## Modelo de Negocio

### Freemium
- Free: 100 notas/mes, 1GB storage, búsqueda básica
- Pro ($9/mes): Ilimitado, búsqueda avanzada, integraciones
- Team ($15/usuario/mes): Colaboración, admin, SSO

### Métricas Clave
- Activación: Primera nota creada en <2 minutos
- Retención: D7 >40%, D30 >25%
- Engagement: >5 notas/semana por usuario activo
- Conversión Free→Pro: >3% en 3 meses
